Course Overview

Welcome to AI 520 - Natural Language Processing for Artificial Intelligence

Course Description

This course provides students with foundational knowledge and understanding of natural language processing (NLP) methods and strategies. Students acquire knowledge to evaluate the strengths and weaknesses of various NLP technologies and frameworks utilizing NLP toolkits to create practical applications.

Key Topics Covered

  • Text Classification
  • Language Modeling
  • Speech Tagging
  • Parsing
  • Semantics
  • Text Summarization
  • Machine Translation
  • Conversational AI

Course Details

Credits: 3 Graduate Credits
Grading Type: Decimal
Prerequisites: Internet access required
Format: All assignments in Microsoft Word-compatible formats

Important Note

All written assignments must follow APA style guidelines. See the library's APA Style Guide tutorial for resources.

Learning Outcomes

What you'll achieve by the end of this course

Upon Course Completion, Students Will:

  • Explain fundamental NLP concepts
  • Integrate researched current and future uses of conversational AI
  • Apply current tools for text classification
  • Evaluate deep learning algorithms used in NLP
  • Evaluate tools and services for common use cases for text analysis and conversational AI
  • Design and implement an AI solution using NLP

Team Project

Research and Design an NLP Model - Create a Conversational AI or Text Generator

Project Overview

Students research NLP models and use cases to create a conversational AI or text generator that solves industry problems. The project must address current or future organizational challenges using appropriate tools introduced in Hands-On Skills exercises.

Project Components (200 Points Total)

  • Proposal: 30 points
  • Progress Report: 70 points
  • Final Report (6-7 pages): 70 points
